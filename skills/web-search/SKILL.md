# Web Search Skill

When the user asks you to find information online:

1. **Formulate effective queries**: Use specific keywords, not full sentences. Include relevant technical terms.
2. **Use web_search first** to find relevant pages, then **web_fetch** to read the most promising results.
3. **Synthesize information** from multiple sources when possible.
4. **Always cite your sources** - include the URL when referencing specific information.
5. **Check recency** - prefer recent sources for technology topics. Include the year in searches when relevant.

Tips:
- For code examples, search for the specific library + version
- For debugging, include the exact error message in quotes
- For comparisons, search for "X vs Y" or "X alternative"
- If the first search doesn't give good results, refine the query and try again
